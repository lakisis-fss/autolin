# 05_ 자동 물약(Auto-Potion) 제어 로직 테스트 보고서

## 개요
HP 인식 데이터(`check_hp.py`)를 기반으로, 설정된 임계값(20%) 이하일 때 물약 단축키(F1)를 자동으로 눌러주는 제어 루프를 구현하고 검증한 결과입니다.

---

## 1. 구현 상세 (`auto_potion.py`)
*   **제어 로직**: 
    *   0.2초 간격(랜덤 지터 포함)으로 실시간 HP 모니터링.
    *   HP <= 20.0% 달성 시 `F1` 키 입력 발생.
*   **입력 기술**: `pydirectinput`을 사용하여 하드웨어 인터럽트 방식에 가까운 키 입력 시뮬레이션 (DirectX 호환).
*   **스텔스 최적화**:
    *   **랜덤 딜레이**: 체크 간격 및 키 누름 시간(0.05s~0.15s)을 매회 무작위로 설정하여 매크로 탐지 회피.
    *   **쿨타임 관리**: 중복 입력을 방지하기 위해 1.0초의 내부 쿨타임 적용.

## 2. 테스트 결과
*   **인식 연동**: `Current HP: 100.0%` 수치를 실시간으로 읽어와 제어 루프에 성공적으로 전달함.
*   **동작 검증**: 
    *   HP가 100%인 상태에서는 입력이 발생하지 않음을 확인.
    *   임계값을 임시로 올려서 테스트 시, 정상적으로 `[ALERT] HP Low! Pressing f1...` 로그와 함께 키 입력이 시뮬레이션됨을 확인.
*   **안정성**: `Ctrl+C`를 통해 안전하게 종료 가능한 예외 처리 로직 작동 확인.

## 3. 결론 및 향후 계획
### 3.1 기술적 결론
화면 인식 기반의 **피드백 제어 시스템(Feedback Control System)**이 안정적으로 작동합니다. `pydirectinput`을 통해 리니지 클래식 클라이언트에도 정상적으로 명령이 전달됨을 확증했습니다.

### 3.2 향후 확장
*   **다중 임계값**: HP 50%일 때는 일반 물약, 20%일 때는 강력한 물약을 먹는 등 다단계 로직 추가 가능.
*   **전투 자동화**: HP 체크 루프를 메인 사냥 루프(몬스터 탐지 및 타겟팅)와 통합하여 완전 자동화 프로토타입 제작 예정.
